# Restaurant Billing System (Java Console Project)

## 📌 Description
This is a console-based Restaurant Billing System built in Java.  
It allows users to manage menu items, place orders, and generate itemized bills with GST calculation.

---

## 🚀 Features
- Add menu items dynamically
- Remove menu items dynamically
- Display menu
- Place orders with quantity
- Calculate subtotal
- Apply GST (5%)
- Generate itemized receipt

---

## 🛠️ Technologies Used
- Java
- OOP (Classes & Objects)
- Collections (ArrayList, HashMap)
- Scanner (Console input)

---

## 📂 Project Structure

RestaurantBillingSystem/
│
├── Menu-Item.java
├── Restaurant.java
└── Main-Restaurant.java


---

## ▶️ How to Run

1. Open project in IntelliJ IDEA / Eclipse / VS Code
2. Compile all Java files
3. Run `Main.java`
4. Use console menu to interact
